# Project Roro

Offline AI Voice Assistant (Jarvis-style)

---

## 🚀 Features

- 🎤 Voice Input (Microphone + VAD)
- 🧠 Speech-to-Text (Whisper)
- 🤖 Local LLM (TinyLlama via llama.cpp)
- 🔊 Text-to-Speech (pyttsx3)
- ⚡ Fully Offline (No API required)

---

## 🧠 Pipeline

Voice → STT → LLM → TTS
